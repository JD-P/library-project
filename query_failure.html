How to use the library search:

Search by availability accepts the following terms:

True, true, 1
False, false, 0

The difference between partial and exact matching search:

A partial match means that when you search any results that *contain* your
query will be returned. For example if you search by author for 'Isaac' you
will get any author named Isaac such as Isaac Asimov. By contrast, if you
do an exact match for Isaac you will recieve few or no results because Isaac
is only a first name and the author field usually includes two.

How to save the results of a book lookup:

Right click on the background of the page and use the option "save page as".
On mobile there should be a a similar option that varies in location based on what
phone you are using and what web browser.
